尺寸单位与响应式布局示例（标题随视口使用 clamp 自适应）

绝对尺寸（px）

下面的容器宽度被限制为 300px，而内部盒子使用 width:1000px（绝对像素）。当父容器或视口更窄时，会出现横向滚动。

width: 1000px
<div class="wrap-300"> <div class="box-px">width: 1000px</div> </div> /* .box-px { width:1000px; } */

设置最大宽度（max-width）

  • max-width 用于设置元素的最大宽度。
  • 当浏览器窗口或父容器小于固定宽度（如 1000px）时，固定宽度会导致横向滚动。
  • 使用 max-width:100% 能让元素在小屏上缩小以适配容器，改善小窗口体验。
width:1000px; max-width:100%
/* 对比：仍声明 width:1000px，但加上 max-width:100% */ .box-max { width:1000px; max-width:100%; }

相对尺寸（%、em、rem）

百分比相对于父元素；em 相对于当前元素继承的字体大小；rem 相对于根元素（html）的字体大小。

width:50%
父容器字体 20px
em 字号：1.5em → 30px
rem 字号：1.5rem（若根为 16px → 24px）
.box-percent-parent { width:80%; max-width:600px; } .box-percent { width:50%; } .parent-20 { font-size:20px; } .child-em { font-size:1.5em; } /* 1.5 × 20 = 30px */ .child-rem { font-size:1.5rem; }/* 1.5 × 根字体大小 */

响应式尺寸（vw/vh、min()、clamp()、媒体查询）

width:50vw（随视口宽度的一半变化）
width:min(90vw, 600px)

上方第二个盒子使用 min(90vw, 600px)：在大屏不超过 600px，小屏占视口 90% 宽度。

本段标题使用 font-size: clamp(16px, 2vw + 8px, 28px)，在不同屏幕下平滑缩放且限定上下限。

.box-vw { width:50vw; } .box-responsive { width:min(90vw, 600px); } .clamp-title { font-size: clamp(16px, 2vw + 8px, 28px); } @media (max-width:600px) { /* 600px 以下触发，演示媒体查询 */ .box-px { background:#ffdce0; } }

响应式图片（max-width:100%）

给图片添加 max-width:100% 与 height:auto，能确保图片在小屏缩小且保持比例。
